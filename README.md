## Overview
基于Flutter的语雀第三方APP

实现了授权登录，查看热门文章，创建、编辑我/团队文档等基础功能。
> [语雀开发者文档](https://www.yuque.com/yuque/developer/api)
---

## Task
- [x] 用户授权登录
- [x] 热门、推荐、精选文章
- [x] 查看我的、团队的知识库
- [x] 新建、修改文档
- [x] 搜索文章
- [x] 阅读历史
- [x] 暗黑模式
- [ ] 知识库、文章的信息修改与删除
- [ ] 更多维度的搜索
- [ ] Webhook接入
- [ ] 优化编辑页面
- [ ] 分享为链接/微信小程序
